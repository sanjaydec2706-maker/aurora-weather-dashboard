# aurora-weather-dashboard
Aurora Weather Dashboard 🌦️ — A modern, responsive weather dashboard built with HTML, CSS, and JavaScript. Features animated weather visuals, hourly &amp; 10-day forecasts, radar view, AI weather insights, community alerts, location switching, and customizable weather units. ✨
# 🌦️ Aurora Weather Dashboard

A modern and elegant weather dashboard built using HTML, CSS, and JavaScript.

## ✨ Features

- Dynamic weather hero section
- Hourly forecast
- 10-day forecast
- Radar visualization
- AI Weather Insights
- Community alerts
- Location switching
- Celsius/Fahrenheit toggle
- Wind speed conversion
- Responsive design
- Modern UI animations


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- SVG Icons
- Responsive Design

## 🚀 Run Locally

1. Download the repository
2. Open `index.html` in your browser

No installation required.
